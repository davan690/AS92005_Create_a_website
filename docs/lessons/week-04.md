# Week 4 — Visualizing Data

## Lesson 10: Introduction to `ggplot2`

**Goal:** Turn data into a chart.

**Activity:**
- Teacher code-along creating simple bar chart.
- Use class dataset example (e.g., monthly rainfall or top categories).

**Teacher checks:**
- Chart renders with correct axes.
- Students can identify chart type and what it communicates.

---

## Lesson 11: Customizing Visualizations

**Goal:** Make charts clear and professional.

**Activity:**
- Students add chart title, axis labels, and color changes.
- Optional: reorder categories or improve legend naming.

**Teacher checks:**
- Label clarity and readability.
- Students justify one visual design choice.

---

## Lesson 12: Mini-Project Presentation

**Goal:** Bring all Phase 1 skills together.

**Activity:**
- Students polish a 3-page practice site.
- Verify theme, text, data table, and chart all render cleanly.
- Quick pair walkthroughs for feedback.

**Teacher checks:**
- Fully working 3-page prototype.
- No blocking YAML/chunk errors before end of week.
