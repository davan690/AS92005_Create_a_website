# Course Resources

This folder stores reusable course-development assets.

## Structure

- `printables/` — generated printable outputs (PDF/HTML handouts)
- `templates/` — reusable planning and evidence templates
- `notes/` — teacher notes, moderation notes, and iteration records

## Suggested workflow

1. Build the website and printables with `./build-course.ps1`.
2. Keep final student/teacher handouts in `resources/printables/`.
3. Add new reusable artifacts to `templates/` over time.
