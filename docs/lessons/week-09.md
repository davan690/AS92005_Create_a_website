# Week 9 — Trialling and Refinement

## Lesson 25: Peer Trialling Session A

**Goal:** Gather user feedback for higher-grade criteria.

**Activity:**
- Pair students (A trials B’s site while B records feedback).
- Focus: navigation clarity, readability, data explanation quality.

**Teacher checks:**
- Feedback is specific and actionable.

---

## Lesson 26: Peer Trialling Session B

**Goal:** Complete reciprocal user testing.

**Activity:**
- Swap roles (B trials A’s site).
- Record findings in testing table with priorities.

**Teacher checks:**
- Each student has complete peer-trial evidence.

---

## Lesson 27: Implementing Feedback

**Goal:** Improve fitness for purpose.

**Activity:**
- Apply trial feedback directly to site and content.
- Document what changed and why in testing/process evidence.

**Teacher checks:**
- Evidence of iteration is visible in final product.
