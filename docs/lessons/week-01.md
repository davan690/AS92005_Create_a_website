# Week 1 — The Paradigm Shift (HTML to Markdown)

## Lesson 1: Welcome to Quarto & RStudio

**Goal:** Understand what a Static Site Generator is.

**Activity:**
- Tour the RStudio (or VS Code) interface.
- Students create a single `Hello World` `.qmd` file.
- Render to HTML and observe source-to-output workflow.

**Teacher checks:**
- Every student successfully renders one page.
- Students can explain difference between source (`.qmd`) and output (`.html`).

---

## Lesson 2: Mastering Markdown

**Goal:** Replace HTML tags with Markdown syntax.

**Activity:**
- Provide a plain-text handout.
- Students add headings, bold, italics, bullet/numbered lists, and links.
- Render and compare visual output.

**Teacher checks:**
- Correct syntax for headers, emphasis, and lists.
- Students fix at least one Markdown error independently.

---

## Lesson 3: Images and Links

**Goal:** Add media and external connections.

**Activity:**
- Students source one image and save locally in project folder.
- Embed image with `![alt text](path)`.
- Add links to two websites and test they open correctly.

**Teacher checks:**
- Image path works locally.
- Links are clickable and include meaningful link text.
