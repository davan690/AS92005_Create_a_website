# Week 6 — Development Sprint 1

## Lesson 16: Workshop — Build Core Site Structure

**Goal:** Establish complete page framework.

**Activity:**
- Build all required pages and navigation links.
- Add placeholder headings/content blocks per page.

**Teacher role:** Technical lead for YAML/nav debugging.

**Checkpoint:**
- Full page scaffold complete.

---

## Lesson 17: Workshop — Core Content Drafting

**Goal:** Populate pages with initial content.

**Activity:**
- Draft introduction/context text.
- Add first data interpretation paragraph(s).
- Insert at least one image/visual support element.

**Teacher role:** Feedback on clarity for target audience.

**Checkpoint:**
- At least 50% content drafted.

---

## Lesson 18: Workshop — Data Integration Baseline

**Goal:** Embed and validate first data output.

**Activity:**
- Load dataset and render one table or chart in project site.
- Check for chunk and path errors.
- Add one entry in Testing Table/development log.

**Teacher role:** Debug R chunk failures.

**Checkpoint:**
- One working computational element per student.
