# Week 2 — Structuring the Site

## Lesson 4: The `_quarto.yml` File

**Goal:** Understand website configuration.

**Activity:**
- Convert single page into Quarto website project.
- Open `_quarto.yml` and identify key sections (`project`, `website`, `format`).
- Demonstrate YAML indentation sensitivity.

**Teacher checks:**
- Students can describe what `_quarto.yml` controls.
- Students fix one intentional indentation error.

---

## Lesson 5: Navigation Menus

**Goal:** Create a multi-page site.

**Activity:**
- Create three pages: Home, About, Data.
- Add navbar entries in `_quarto.yml`.
- Render and verify each page link works.

**Teacher checks:**
- Navigation appears on every page.
- No broken links between pages.

---

## Lesson 6: Styling with Bootswatch

**Goal:** Apply themes without writing raw CSS.

**Activity:**
- Explore built-in themes (e.g., Minty, Slate, Quartz).
- Apply one theme in YAML.
- Compare readability and audience suitability.

**Teacher checks:**
- Students can switch themes successfully.
- Students justify one theme choice for audience.
