# 10-Week Lesson Plans (AS92005 + Quarto)

This folder contains day-by-day lesson plans for a 10-week term (3 lessons per week, 30 lessons total).

## Structure

- `week-01.md` to `week-04.md` — Phase 1: Skill Building
- `week-05.md` to `week-10.md` — Phase 2: AS92005 Assessment

## Sequence Overview

| Week | Lessons | Focus |
|---|---:|---|
| 1 | 1–3 | Paradigm shift: HTML to Markdown |
| 2 | 4–6 | Quarto website structure + styling |
| 3 | 7–9 | Intro R chunks + loading/displaying data |
| 4 | 10–12 | Visualization + mini-project integration |
| 5 | 13–15 | Assessment planning and design |
| 6 | 16–18 | Sprint 1 development workshops |
| 7 | 19–21 | Sprint 2 development workshops |
| 8 | 22–24 | Sprint 3 development workshops |
| 9 | 25–27 | Peer trialling and refinement |
| 10 | 28–30 | Final render, publish, submit |

## Notes for use

- Plans are designed for local rendering and static deployment (`docs/`) workflow.
- Keep class examples aligned to your environmental datasets:
  - `data/environment_nz.csv`
  - `data/waste_recycling_nz.csv`
- Ask students to update their testing/development log at least once per week.
