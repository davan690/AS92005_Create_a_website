# Week 5 — AS92005 Planning and Design

## Lesson 13: Unpacking the Assessment Brief

**Goal:** Understand AS92005 requirements.

**Activity:**
- Review project brief and deliverables.
- Students choose data-driven topic and dataset option.
- Start personal project scope statement.

**Teacher checks:**
- Topic is feasible in timeframe.
- Dataset selected and accessible.

---

## Lesson 14: Target Audience and Implications

**Goal:** Meet Achieved/Merit planning criteria.

**Activity:**
- Students define audience, purpose, and communication approach.
- Write short audience implications paragraph.

**Teacher checks:**
- Audience is specific.
- Design implications are practical and observable.

---

## Lesson 15: Wireframing and Project Setup

**Goal:** Plan site architecture.

**Activity:**
- Draw wireframes for minimum 4 pages.
- Create official AS92005 project folder and initial Quarto setup.

**Teacher checks:**
- Wireframes match navigation plan.
- Base project renders without errors.
